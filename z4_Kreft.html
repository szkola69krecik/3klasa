1)kiedy powstał JavaScript (podaj datę).

JavaScript jest obiektowym skryptowym językiem programowania. Stworzony został przez firmę Netscape w 1995 roku.


2)dlaczego jest to język skryptowy

JavaScript to język skryptowy znany ze swojej interpretacji podczas działania, dynamicznego typowania, łatwości użycia i specjalizacji w obszarze przeglądarek internetowych. Popularny w tworzeniu interaktywnych interfejsów użytkownika.


3)gdzie jest wykonywany JS (klient lub serwer)

Po Stronie Klienta: W przeglądarkach internetowych, JavaScript jest wykonywany po stronie klienta. Skrypty JavaScript osadzone są w kodzie HTML strony internetowej i są interpretowane przez silnik JavaScript wbudowany w przeglądarkę. Służy głównie do dostarczania interaktywności na stronach internetowych, obsługi zdarzeń użytkownika, manipulacji elementami DOM (Modelu Obiektowego Dokumentu) i komunikacji z serwerem za pomocą technologii takich jak AJAX.


4)dlaczego jest JS to bezpieczne narzędzie

JavaScript jest stosunkowo bezpieczny dzięki mechanizmom takim jak "sandboxing" i polityce samego pochodzenia w przeglądarkach, co ogranicza dostęp do zasobów systemu i różnych domen. Niemniej jednak, utrzymanie bezpieczeństwa wymaga świadomości zagrożeń i praktyk programistycznych.


5)czy jest to język obiektowy , jeśli tak to dlaczego

Tak, JavaScript jest językiem obiektowym. Posiada obiekty i prototypy, wprowadzenie klas w ECMAScript 6, umożliwia enkapsulację, polimorfizm i dziedziczenie. Chociaż może być używany w różnych stylach programowania, jego cechy i mechanizmy są charakterystyczne dla języków obiektowych.

Framework to:

7)Framework to struktura, narzędzia i zasady pomagające programistom w tworzeniu aplikacji. Zapewniają gotowe komponenty, wzorce projektowe, automatyzację zadań i zasady konwencji nazewniczych, ułatwiając rozwój oprogramowania. Przykłady to Angular, React, Vue.js dla stron internetowych czy Django, Flask dla aplikacji webowych.


8)Do aplikacji mobilnych można użyć Framework

Aplikacje mobilne można tworzyć przy użyciu różnych frameworków. Przykłady to React Native, Flutter, Xamarin, Ionic i NativeScript. Każdy z tych frameworków ma swoje zalety, a wybór zależy od preferencji programisty i wymagań projektu.


9)Aplikacje responsywne to:

Aplikacje responsywne to aplikacje, które dynamicznie dostosowują się do różnych rozmiarów ekranów i urządzeń. Oznacza to, że interfejs użytkownika jest elastyczny i zoptymalizowany, aby zapewnić wygodne korzystanie zarówno na dużych ekranach komputerów, jak i na mniejszych ekranach urządzeń mobilnych. W rezultacie aplikacje responsywne oferują spójne doświadczenie użytkownika niezależnie od używanego urządzenia.

12)Opisz następującą instrukcję window.close()

window.close() to instrukcja JavaScript używana do zamknięcia bieżącego okna przeglądarki. Jeśli jest wywoływana, próbuje zamknąć to okno. Z uwagi na zasady bezpieczeństwa, przeglądarki mogą blokować automatyczne zamknięcie okna, wymagając interakcji użytkownika.
